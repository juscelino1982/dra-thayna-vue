# Configuração Git Local

Este projeto usa configuração git **LOCAL** ao invés de global, pois há outro git nessa máquina.

## Configuração Atual

```bash
git config --local user.email "juscelino1982@gmail.com"
git config --local user.name "Juscelino"
```

## Verificar configuração

```bash
git config --local user.email
git config --local user.name
```

## Nota Importante

Sempre use `git config --local` ao invés de `git config --global` neste projeto.
